Os Blocos de Construção: Pronúncia

A pronúncia portuguesa tem algumas regras únicas. Compreender estas letras-chave é o primeiro passo para soar como um falante nativo.

A Letra C

  • KAntes de 'a', 'o', 'u' soa a 'k'. Ex: carro, cor.
  • SAntes de 'e', 'i' soa a 's'. Ex: cenoura, cidade.
  • SHComo 'ch' soa a 'sh'. Ex: chuva.

A Letra S

  • ZEntre duas vogais soa a 'z'. Ex: casa.
  • JAntes de uma consoante sonora (b, d, g, l, m, n) soa como o 's' em 'pleasure'. Ex: Lisboa.
  • SHNo final de uma palavra ou antes de uma consoante surda (c, f, p, q, t) soa a 'sh'. Ex: mesas, escola.

A Letra X

Esta letra é um camaleão! O seu som muda dependendo da palavra.

  • SH: peixe
  • Z: exemplo
  • KS: táxi
  • S: aproximar

A Melodia: Acentos e Vogais

Os acentos não são apenas para decoração; são vitais para a pronúncia e o significado, alterando o som de uma vogal.

´

Agudo

Marca a sílaba tónica e um som de vogal aberta. Ex: pé, avó.

^

Circunflexo

Marca a sílaba tónica e um som de vogal fechada. Ex: mês, avô.

~

Til

Cria um som nasal, como se o ar passasse pelo nariz. Ex: pão, mãe.

`

Grave

Usado apenas em 'a' para mostrar a contração de dois 'a's (a + a = à). Indica um som aberto.

Gramática Essencial: Substantivos e Verbos

Compreender o género e os dois verbos "ser/estar" é fundamental para construir frases corretas.

Distribuição de Género dos Substantivos

Em português, cada substantivo é masculino ou feminino. A terminação de uma palavra é muitas vezes uma pista. Este gráfico mostra a tendência geral das terminações.

Ser vs. Estar

Ambos significam "to be", mas não são intercambiáveis. Ser é para características permanentes, enquanto Estar é para estados temporários.

SER

(Permanente)

  • Identidade
  • Origem
  • Profissão
  • Tempo
  • Características

ESTAR

(Temporário)

  • Localização
  • Humor
  • Condição
  • Tempo (clima)
  • Ações

Exemplo: "Ela é bonita" (característica). "Ela está bonita" (estado temporário).

Poder Verbal: Conjugações Regulares

A maioria dos verbos segue um padrão previsível com base na sua terminação: -ar, -er ou -ir. Aprenda o padrão e poderá conjugar centenas de verbos!

Pronome -AR (falar) -ER (comer) -IR (partir)
Eu falo como parto
Tu falas comes partes
Ele/Ela/Você fala come parte
Nós falamos comemos partimos
Eles/Elas/Vocês falam comem partem
