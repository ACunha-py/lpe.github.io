The Building Blocks: Pronunciation

Portuguese pronunciation has some unique rules. Understanding these key letters is the first step to sounding like a native speaker.

The Letter C

  • KBefore 'a', 'o', 'u' it sounds like 'k'. Ex: carro, cor.
  • SBefore 'e', 'i' it sounds like 's'. Ex: cenoura, cidade.
  • SHAs 'ch' it sounds like 'sh'. Ex: chuva.

The Letter S

  • ZBetween two vowels it sounds like 'z'. Ex: casa.
  • JBefore a voiced consonant (b, d, g, l, m, n) it sounds like the 's' in 'pleasure'. Ex: Lisboa.
  • SHAt the end of a word or before a voiceless consonant (c, f, p, q, t) it sounds like 'sh'. Ex: mesas, escola.

The Letter X

This letter is a chameleon! Its sound changes depending on the word.

  • SH: peixe
  • Z: exemplo
  • KS: táxi
  • S: aproximar
